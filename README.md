#  Settings-visionOS
SwiftUI recreation of the visionOS Settings app.

## Information
- This project is a recreation of Apple's default and current Settings app (as of visionOS 1.0 [21N305]) for Apple Vision Pro **Simulator** using Swift and SwiftUI.
- All features within the app are only simulations and have no effect on the device outside of the app.

## Usage
Open the project in Xcode and run with either a visionOS Simulator of Apple Vision Pro or a physical Apple Vision Pro with Developer Mode enabled as a destination.
